BugPatrol

This bug tracking web application enhances project management by allowing users to track and address bugs efficiently. Built with Express, Node.js, React, and SQL, it integrates AWS Amplify and Cognito for secure user authentication and data integrity. The system features a backend server managing API endpoints and data synchronization with an AWS MySQL database. It employs Redux for state management and Bootstrap for a refined user interface, enabling functionalities like account creation, project initiation, and role-specific member inclusion. Users can create, assign, and oversee bugs, fostering effective team collaboration and project control in a streamlined and intuitive environment.
